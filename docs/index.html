Debian Packages for Koha

This is an apt repository for Koha and its dependencies.

To add this repository in order to install the current stable release of Koha, add the line
deb http://debian.koha-community.org/koha stable main
to /etc/apt/sources.list.

For complete instructions on how to use the Koha packages, including how to install the development or previous stable versions of Koha, see https://wiki.koha-community.org/wiki/Debian.
